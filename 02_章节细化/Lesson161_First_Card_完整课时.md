# Lesson 161：First Card

## 课程信息

- 级别：L0 复现扩展
- 年龄：4 周岁
- 时长：8 到 10 分钟
- 画面数：8
- 核心句：`First card.`
- 目标词：first, card, start
- 听力目标：能听懂 first, card, start 的安全日常场景
- 口语目标：能跟读 `First card.` 或其中 1 个目标词

## 课时流程

1. Warm-up：三张无字故事卡放在低矮桌上
2. Learn：认识 first, card, start。
3. Story：Nia、Lumo 和 Bobo 在安全情境中练习 `First card.`。
4. Tap：点击核心物品或动作。
5. Say：孩子跟读 `First card.` 或其中 1 个目标词。
6. Wrap-up：三位角色用本课核心句温和收束。

## 互动题

1. `Tap first.` 正确答案：First
2. `Tap card.` 正确答案：Card
3. `Tap start.` 正确答案：Start

## 家长陪伴提示

- 顺序主题只用无字故事卡，不做阅读测试。
- 可以指第一张卡说 `first`。
- 不表现文字、复杂规则、输赢或压力测验。
