# Lesson 50：Clean Up Toys

## 课程信息

- 级别：L0 复现扩展
- 年龄：4 周岁
- 时长：8 到 10 分钟
- 画面数：8
- 核心句：`Clean up, please.`
- 目标词：clean up, toys, box
- 听力目标：听到 toys / box / clean up 能指出玩具、盒子或整理动作
- 口语目标：能模仿 `Clean up, please.`，进阶说 `Toys in the box.`

## 课时流程

1. Warm-up：地垫上有少量大玩具。
2. Point：指出 toys 和 box。
3. Story：三位角色把玩具放进盒子。
4. Act：整理一个真实玩具。
5. Tap：点击 toys、box、clean up。
6. Wrap-up：地面变干净。

## 互动题

1. `Tap toys.` 正确答案：Toys
2. `Tap box.` 正确答案：Box
3. `Tap clean up.` 正确答案：Clean up

## 家长陪伴提示

- 每次只整理 1 到 3 个大玩具。
- 避免小颗粒玩具，先用球或大积木。
- 孩子说 `box` 或 `clean up` 就算完成。
