# 儿童英语绘本课程项目

这是一个面向 4 岁儿童的英语听说启蒙绘本项目。目标不是复刻 RAZ，而是参考其“分级、可听、可读、可互动、可追踪”的产品思路，做一套原创的家庭可用课程。

## 当前定位

- 年龄：4 周岁
- 重点：听力输入、跟读、简单口语输出
- 形式：每课不超过 10 幅画，优先漫画分镜
- 互动：点击听音、点选理解、跟读录音、简单动效
- 版权原则：不直接使用艾莎公主、奥特曼、RAZ 原文或原图；使用原创“冰雪公主/光能英雄”等相似类型但独立的角色设定

## Lesson 31 金标准课

《Shoes On, I'm Ready!》是“把绘本英语说进生活”的第一节可测试标杆课。它把一次完整体验拆成 7 个阶段、12 个步骤：故事情境、点图找鞋、动作理解、儿童自然输出、离屏穿鞋、回来展示和下一次真实出门复现。故事与展示段会连续播放，在需要孩子找、做、说时才停下；页面使用稳定 3:2 舞台、阶段内微进度、即时角色回应和默认收起的家长提示。现已实现画面热点、动作完成方式、可选本地录音与删除、离屏安静计时、三种无失败结果、刷新续课和本机完成记录。

麦克风不会在页面加载时自动申请。开始页可选择提前授权；到“我来说”时，页面会先说明录音用途和角色回应，再由家长选择“打开麦克风”或“我说完了，不录音”，只有前者会触发系统权限。页面区分已允许、已拒绝、无设备、设备占用、非安全页面和浏览器不支持，所有异常都保留直接跟说路径。录音只生成当前页面内存中的临时回放地址，不写入本地进度、不上传；关闭或刷新页面后录音消失。标杆课使用兼容性更好的 MP3，引导音频位于 `assets/audio/benchmark31/`。

建议通过本地服务器预览，以保证音频和麦克风权限行为与正式网页一致：

```bash
cd /Users/hik/Desktop/workspace/儿童英语绘本
python3 -m http.server 4173 --directory .
```

然后打开 `http://127.0.0.1:4173/app/static/benchmark.html`。课程库首页顶部也有标杆课入口。
